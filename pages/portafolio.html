Conceptos Fundamentales de CSS

A continuación se explican los principales conceptos de CSS aplicados en este proyecto, con ejemplos de código y descripción de su uso práctico. Esta sección está organizada con CSS Grid.

Colores, Fuentes y Fondos

Permiten personalizar la apariencia visual de los elementos mediante texto, tipografía y color de fondo.

  • color: color del texto
  • background-color: fondo del elemento
  • font-family: tipo de fuente
  • font-size: tamaño del texto
  • font-weight: grosor de la fuente
body {
        color: #222222;
        background-color: #f4f6f8;
        font-family: Arial, sans-serif;
        font-size: 16px;
        font-weight: normal;
    }

Bordes, Margen y Relleno

Controlan el espacio alrededor y dentro de cada elemento, así como su contorno visual y dimensiones.

  • border: grosor, estilo y color del borde
  • border-radius: esquinas redondeadas
  • margin: espacio externo al elemento
  • padding: espacio interno del elemento
  • width / height: dimensiones
.tarjeta-proyecto {
  border: 2px solid #c8e6c9;
  border-radius: 8px;
  margin-bottom: 1.8rem;
  padding: 1.5rem;
  width: 100%;
}

Modelo de Caja

El modelo de caja describe cómo el navegador calcula el espacio total que ocupa cada elemento HTML, con cuatro capas:

  • Content: el contenido real (texto, imagen)
  • Padding: espacio entre contenido y borde
  • Border: línea que rodea el padding
  • Margin: espacio externo entre elementos
* {
  box-sizing: border-box;
  /* padding y border se incluyen
     dentro del width total del
     elemento, evitando desborde */
}

Propiedad Display

Controla cómo se comporta y se renderiza un elemento en el flujo normal del documento.

  • block: ocupa toda la línea (div, p, h1)
  • inline: fluye dentro del texto (span, a)
  • inline-block: mezcla de ambos comportamientos
  • flex: activa el modelo Flexbox
  • grid: activa el modelo Grid
  • none: oculta el elemento completamente
nav {
  display: flex; /* fila de enlaces */
}
.etiqueta {
  display: inline-block; /* badges */
}
.grid-css-conceptos {
  display: grid; /* cuadrícula */
}

Posicionamiento

La propiedad position controla cómo se ubica un elemento, usando las propiedades top, right, bottom y left.

  • static: posición por defecto, flujo normal
  • relative: desplazado desde su lugar original
  • absolute: relativo al ancestro posicionado
  • fixed: fijo respecto a la ventana del navegador
  • sticky: fijo al hacer scroll hasta un límite
.figure-flotante {
  position: relative;
  float: right;
  margin-left: 1.5rem;
}
nav {
  position: sticky;
  top: 0;
  z-index: 100;
}

Flexbox

Modelo de diseño unidimensional que organiza elementos hijos en fila o columna con control preciso del alineamiento.

  • display: flex: activa el contenedor flex
  • flex-direction: dirección (row / column)
  • justify-content: alineación en eje principal
  • align-items: alineación en eje cruzado
  • flex-wrap: permite saltos de línea
  • gap: espacio uniforme entre elementos
nav {
  display: flex;
  flex-wrap: wrap;
  justify-content: center;
  align-items: center;
  gap: 12px;
}

CSS Grid

Modelo de diseño bidimensional que organiza elementos en filas y columnas simultáneamente, ideal para maquetaciones complejas.

  • display: grid: activa el contenedor grid
  • grid-template-columns: define las columnas
  • grid-template-rows: define las filas
  • gap: espacio entre celdas
  • auto-fit + minmax: columnas responsivas
  • grid-column / grid-row: posición de un elemento
.grid-css-conceptos {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(280px, 1fr));
  gap: 1.2rem;
}

Habilidades Adquiridas

Código Limpio y Semántico

Aplicación de buenas prácticas en la escritura de HTML: indentación correcta, uso apropiado de etiquetas semánticas y comentarios descriptivos para documentar cada sección del código fuente.

Diseño Visual con CSS

Uso de hojas de estilo para crear páginas con identidad visual propia: paleta de colores coherente, tipografía legible, espaciados adecuados y efectos interactivos que mejoran la experiencia del usuario.

Navegación y Rutas Relativas

Creación de menús de navegación funcionales con rutas relativas correctas entre múltiples páginas de un mismo sitio web, incluyendo enlaces internos, externos y de anclaje a secciones específicas.

Diseño Responsivo

Aplicación de media queries para adaptar el diseño a diferentes tamaños de pantalla, asegurando una experiencia de usuario óptima tanto en dispositivos móviles como en pantallas de escritorio.
